# Grocery Retail Analytics Dashboard

A data analytics project built using Power BI to explore sales performance, product trends, and customer behavior in a grocery retail environment.
This dashboard provides insights to support better business decisions through interactive and visual analysis.

<h3>📦Technologies Used</h3>
- Power BI 

<h3>🥁Features</h3>
- Sales Overview Dashboard<br>
- Customer Analytics Dashboard<br>
- Interactive Filters

<h3>👨‍🍳The Process</h3>
<b>Data Preparation</b><br>
- Cleaned and transformed raw grocery retail data<br>
- Handled missing values and ensured data consistency<br>

<b>Data Modeling</b><br>
- Designed a star schema with fact and dimension tables<br>
- Established relationships between tables<br>

<b>DAX Calculations</b><br>
Created key metrics such as:<br>
- Total Sales<br>
- Average Order Value (AOV)<br>
- Return Purchase Rate (RPR)<br>

<b>Dashboard Design</b><br>
- Built two main pages: Overview and Customers<br>
- Applied consistent color themes and layout<br>

<b>Insight Generation</b><br>
- Identified customer behavior patterns<br>
- Highlighted business opportunities and limitations<br>

<h3>🚦Running the Project</h3>
1. Download the .pbix file from this repository<br>
2. Open it using Power BI Desktop<br>
3. Ensure all data sources are properly loaded<br>
4. Use slicers and filters to explore the dashboard interactively
